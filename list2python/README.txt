LISTA 2:

Rozwiązaniem zadania na liście 2 jest plik 'task1.py', jest zadanie na ocenę 5. DO jego uruchomienia wymagany jest 
python 3.7 oraz biblioteka bitarray, do pobrania biblioteki należy posłużyć się poleceniem `pip install bitarray`.

Program uruchamia się poleceniem:
    `python arithmetic.py [-d] -i [input_file] -o [output_file]`
znaczenie flag:
    -d - dekompresja z pliku wejścowego, plik wyjściowy to plik nieskompresowany, w przeciwnym razie następuje kompresja
    -i <arg> - oznaczenie pliku wejściowego
    -o <arg> - oznaczenie pliku wyjściowego
